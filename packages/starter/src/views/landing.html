Let's get started!

Monoblogger is a framework that streamlines the creation of custom Blogger templates.

  1. 1 Clear any leftover widgets on the Layout tab to start from a clean slate.
  2. 2 Consult the docs to learn how views, data, and the build pipeline fit together.
  3. 3 Compose your layout from views, style it with Tailwind CSS, and ship a single dist/index.xml.
